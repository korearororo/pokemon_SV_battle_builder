# SV Battle Workspace

포켓몬 SV 실전 개체 등록, 매치업 계산, AI 코치 기능을 웹/모바일로 병렬 개발하기 위한 워크스페이스입니다.

## 디렉토리 구조

- `web/`: Next.js 기반 웹 앱
- `mobile/`: Expo(React Native) 기반 모바일 앱

## 실행 방법

### 웹

```bash
npm install
npm run dev:web
```

### 모바일

```bash
npm install
npm install --workspace mobile
npm run dev:mobile
```

모바일 플랫폼 실행:

```bash
npm run android:mobile
npm run ios:mobile
```

## 웹 주요 기능

- 개체 추가/관리
  - 포켓몬, 특성, 도구, 성격, 테라 타입, EV/IV 입력
  - 기술은 0~4개 등록 가능
  - 동일 이름 수정 저장/신규 저장, 삭제/비우기 지원
- 매치업 계산
  - 왼쪽 기술/오른쪽 기술 기반 동시 계산
  - 우선도 + 스피드 순서 반영
  - 1행동 후 HP를 반영한 2행동 순차 계산
  - 테라스탈 ON/OFF 토글
  - 기믹 태그 및 설명 표시
- AI 코치
  - 채팅형 인터페이스
  - 상태 표시(연결 중/작성 중)
  - 추천 개체를 버튼으로 바로 등록

## 최근 반영 사항

- 웹/모바일 병렬 개발 구조로 리포지토리 정리 (`web`, `mobile`)
- 개체 관리 UX 개선
  - 드롭다운에서 기존 개체 선택/새로 추가
  - 수정/저장, 삭제/비우기 플로우 정리
- 매치업 계산 로직 개선
  - 좌/우 기술 선택 방식 고정
  - 순차 행동 결과 및 HP 비례 위력 반영
  - 테라스탈 토글 가시성 개선
  - 크로스썬더/크로스플레임 등 기믹 표기 강화
- AI 코치 탭 추가 및 응답 상태/자동 스크롤 개선
- UI 전반 한글 인코딩 깨짐 문자열 대규모 복구
